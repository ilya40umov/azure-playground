# Azure Functions (Python) deployed via Terraform

### Commands

#### To get help

```commandline
make help
```

#### Create venv

```commandline
make venv
```

#### To initialize terraform

```shell
make tf-init
```

#### To create/update infra

```shell
make tf-apply
```

#### To send a test request

```commandline
make send-test-request
```

#### To destroy infra

```shell
make tf-destroy
```
